# make lolcat from golang

### How to build and install
1. clone repository
```
$ git clone https://github.com/y-ono366/gololcat.git
```
2. I can now go build; go install and the triology fortune cowsay and gololcat is completed
```
$ cd gololcat
$ go install
```
